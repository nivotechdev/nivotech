## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) from the logo to represent technology and innovation; saturated and vibrant for bold styling.
- Background color: Dark gray (#050505) provides a premium dark mode feel with subtle blue gradients for depth (desaturated from the primary at 20% saturation).
- Accent color: Cyan Tech (#00FFFF), analogous to Electric Blue, with higher brightness and saturation, to highlight key interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headings and short texts to convey a modern tech aesthetic; 'Inter' (sans-serif) for body text to ensure readability in longer sections.
- Use Tailwind CSS for a responsive layout with optimized Core Web Vitals performance.
- Implement scroll-driven animations and header transitions using Framer Motion.
- Incorporate a subtle glow effect on hover for interactive elements to enhance user engagement.
